# ☕ Klasifikasi Jenis Biji Kopi Mentah (Green Bean) dengan CNN

Repositori ini berisi kode sumber untuk aplikasi klasifikasi varietas biji kopi mentah menggunakan algoritma **Convolutional Neural Network (CNN)** dengan arsitektur *Transfer Learning* **MobileNetV2**.

Proyek ini Model pembelajaran mesin di dalam aplikasi ini telah dilatih untuk mendeteksi 4 varietas biji kopi, yaitu:
* Arabika
* Robusta
* Liberika
* Excelsa

## 🛠️ Teknologi & Pustaka yang Digunakan
* **Bahasa Pemrograman:** Python
* **Kecerdasan Buatan (AI):** TensorFlow & Keras
* **Antarmuka Pengguna (Web UI):** Streamlit
* **Pengolahan Citra:** OpenCV, PIL, & Scikit-learn
